# Prodesk IT - Digital Solutions Landing Page

A modern and responsive landing page for Prodesk IT, built as part of Sprint 01.

## 🚀 About the Project

Prodesk IT is a digital solutions company focused on helping businesses grow through technology, web development, SEO, and digital marketing.

This landing page presents the company's services through a clean, modern, and responsive user interface.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Mobile navigation menu
- Dark mode toggle
- Smooth scrolling navigation
- Modern hero section
- About section
- Services section
- Contact call-to-action section
- Responsive service cards
- Social media links
- SEO-friendly meta description

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
